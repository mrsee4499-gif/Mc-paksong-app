🔥 Firebase Configuration

⚠️ ຍັງບໍ່ໄດ້ເຊື່ອມ Firebase — ໃຊ້ Demo Mode

📖 ວິທີໃຊ້ Firebase

ຂັ້ນຕອນ Setup Firebase
  1. ໄປ console.firebase.google.com
  2. ສ້າງ Project ໃໝ່ → "MC Paksong"
  3. ໄປ Project Settings → General → Your apps
  4. ກົດ Web (</>) → Register app
  5. Copy firebaseConfig → ວາງໃສ່ fields ຊ້າຍ
  6. ກົດ "ເຊື່ອມ & ບັນທຶກ" ✅
Enable Firestore & Auth
Build → Firestore Database → Create
Build → Authentication → Email/Password ✓
Storage → Get started ✓
Firestore Rules (ສຳຄັນ)
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
   match /{document=**} {
    allow read, write: if request.auth != null;
   }
  }
}

📖 ວິທີຂໍ API Key CallMeBot

ຂັ້ນຕອນ (ໃຊ້ເວລາ 2 ນາທີ)
  1. ເປີດ WhatsApp → ຄົ້ນຫາ CallMeBot
  2. ສ່ງ: I allow callmebot to send me messages
  3. ລໍຖ້າ reply → ໄດ້ API Key ທັນທີ
  4. Copy API Key → ວາງໃສ່ field ຊ້າຍ
  5. ໃສ່ເບີໂທ +856XXXXXXXXX
  6. ກົດ "ສົ່ງ Test" → ຕ້ອງໄດ້ຮັບ WhatsApp ✅
